1.值传递和地址传递

值传递：传入变量的数值，函数不能改变变量的数值；

地址传递：传入变量的地址，函数可以根据变量地址，访问变量，自然就能改变变量的值；

特别的——数组传递，由于c语言数组和指针的处理方式一样，数组传递就等同于地址传递，函数可以根据地址，访问数组元素，自然就能改变数组元素的值。

值传递只是将变量的内容复制一份而已，函数进行操作的其实是另一个变量，只是另一个变量的值和传递的变量值是相同的。而地址传递是直接把变量的地址传递给函数，这时函数是直接对原来的变量进行操作的。所以值会变化。

举例说吧：值传递就相当于克隆了一个同样的变量，操作的是克隆出来的那个，原来的变量没有变化，但是地址传递是直接对原来的变量进行操作，没有克隆复制新的变量。也可以这样理解：比如要考试了，你有一份复习题，值传递就相当于别人借你的试卷去复印，然后在复印的那份上面做笔记，而地址传递就是别人直接在你原有的试卷上做笔记，没有复印，所以等你拿回来你会发现自己的试卷会有别人的标记和注释。

bit 与 Byte 的区别和联系

区别

一、用途不一样

Bit 意为"位"或"比特"，是计算机bai运算的基础单位；

Byte 意为"字节"，是计算机文件大小的基本计算单位；

二.代表的大小不同

1 bit 就是1位二进制数，比如 1 或者 0；2bit就是2位二进制数，比如：00 / 01 / 10 / 11

1 Byte 就是 1 个字节，1 个字节是由 8 个二进制位组成的。比如1111111 / 00000000 / 10101010 / 10111001 等。

三、代表的内容不一样

Bit 只能代表一个内容，（0或者1，其中的一个）

1 Byte 代表 256 个内容 = 2 的 8 次方。

联系

一个 bit 就是一个二进制数的最小单元，就像我们说我们能够拥有的最小金额的钱就是一分一样，只不过比特是在二进制数中罢了。如果将 8 个 0 或者 1 组合在一起，例如 0110 1010（在中间加一个空格是为了看起来清晰明了），就可以说它们是八个比特或者说一个字节。

1 Byte=8 bit ，不管它们都是全是0、全是1或者是由若干0和若干1混合而成。

位 bit (比特)：存放一位二进制数，即 0 或 1，最小的存储单位。

字节 byte：8个二进制位为一个字节(Byte)，最常用的单位。
